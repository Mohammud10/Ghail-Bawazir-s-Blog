🌴 مدونة غيل باوزير (Ghayl Bawazir Blog)
مدونة ثقافية وإخبارية متكاملة تهدف إلى إبراز التراث والثقافة الخاصة بمدينة غيل باوزير. تم بناء هذا المشروع باستخدام أحدث تقنيات الويب من مايكروسوفت لتوفير تجربة مستخدم سريعة، آمنة، وعصرية.

🚀 التقنيات المستخدمة (Technologies)
إطار العمل: ASP.NET Core MVC 6.0
قاعدة البيانات: MS SQL Server
تقنية ربط البيانات: Entity Framework Core (Code-First Approach)
واجهة المستخدم: HTML5, CSS3, JavaScript, Bootstrap 5
نظام الحماية: Hashing Algorithms (لتشفير كلمات المرور) و Custom Role-based Authorization
✨ المميزات الرئيسية (Key Features)
🔐 نظام صلاحيات متقدم (Roles Management):

المدير (Admin): تحكم كامل بالموقع (إدارة المستخدمين، منح وسحب الصلاحيات، التحكم بالمقالات والتعليقات).
المحرر (Editor): صلاحيات تقتصر على إضافة وتعديل المقالات والأخبار فقط.
الزائر (User): تصفح الموقع، القراءة، التفاعل، وترك التعليقات.
📰 إدارة المحتوى والأخبار:

نظام نشر مقالات ديناميكي مع إمكانية إرفاق الصور.
شريط أخبار متحرك (News Ticker) في الصفحة الرئيسية يعرض آخر المستجدات.
عداد مشاهدات ذكي لكل مقال يزيد مع كل زيارة.
ميزة البحث المباشر في المقالات.
🎨 تجربة مستخدم حديثة:

الوضع الليلي (Dark Mode): زر سلس للتبديل بين الوضع الفاتح والليلي مع حفظ تفضيلات المستخدم.
تصميم متجاوب بالكامل (Responsive) يتناسب مع جميع شاشات الهواتف والأجهزة اللوحية.
👤 صفحات تفاعلية وخاصة:

صفحة "مداد": صفحة شخصية ذكية لعرض النبذات والسير الذاتية (تظهر فقط عند وجود محتوى).
مركز التحميلات: قسم مخصص لرفع ومشاركة الملفات (مثل الـ CV أو التطبيقات) للزوار بروابط مباشرة.
ألبوم الصور (Gallery): قسم مخصص لعرض صور التراث وفعاليات المدينة.
💬 تفاعل الجمهور:

نظام تعليقات مدمج أسفل كل مقال.
أزرار مشاركة سريعة للمقالات عبر وسائل التواصل الاجتماعي (واتساب، فيسبوك، تويتر).
نموذج "اتصل بنا" يستقبل الرسائل ويخزنها مباشرة في لوحة التحكم للإدارة.
